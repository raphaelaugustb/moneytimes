### Moneytimes - Administrate your bills and save your money




#### Setup Project

Clone project

```bash
  git clone https://github.com/raphaelaugustb/moneytimes/
```

Enter on directory

```bash
  cd moneytimes
```

#### Setup back-end

Install depedencies

```bash
  cd backend/moneytimes && mvn clean install package 
```
Run database on container

```bash
  docker compose up
```
Run api

```bash
  mvn spring-boot:run 
```
